去年搞好了博客结果就再也没用过……

发表于 2021-08-10 更新于 2021-08-13 分类于 随笔 ， 图集

去年写了2篇之后就再也没有动过这个博客, 今天重新把它搞起来。架设博客用的 Hexo+NexT 都有了很大的更新, 只能自己从头搞了。废了半天劲终于是把网站重新架起来, 又折腾了点 git, 在 github 仓库里开了一个存放 hexo 源代码的 branch, 这样在其他机子上也可以写博客, 至少拿 macbook 出门的时候又多了点事可以做。

既然重新搞好了, 那总还是得加点新东西的吧, 就在这里简单记录一下这一年都发生了什么吧。

自从去年入门计算之后, 就一直在学习新东西。在换了新的台式机之后, 我目前拿 VMware 里的 Centos 8 虚拟机来当 Windows 下的 terminal 用, 手上的计算资源也从利物浦的 barkla 换成了牛津化学系的 hartree, 学校的 arcus, 还有 UCL 的 young。今年初开始 VASP 推出了 VASP6, 拿到源代码之后就开始沉迷编译 vasp, 前几天还试着给 vasp.6.2.1 加上 Grimme 的 DFT-d4。回顾了去年写的第 1 篇博文里讲装 Linux 虚拟机和用 barla 跑 vasp 遇到的问题感觉自己好傻逼, 要是换现在 vasp 跑不起来, 绝对撸起袖子自己编译算了, 要什么管理员给权限, 当时像天书一般的 bash 脚本现在也能看懂个大概了。去年从组里 Saad 的基于 pymatgen 的作图脚本 fork 出了一份, 发展到现在和现在 Saad 分享的作图脚本已经完成变成不同的东西, 也算是个用 python 的练手机会。理论方面啃完了 Hoffman 的 2 篇讲 Solid state chemistry 的文章, Szabo 的 Modern Quantum Chemisty 也快看完了。目前手上正在做的是一个钙钛矿 project, 不过最近看了点电子结构分析的东西, 打算有空魔改一个 VASP 的代码, 模仿里面的 ELF 模块做个 DORI 分析之类的。前几天把家里的破 Sky hub 换成一个 asus 的 dsl 路由器, 终于有了端口转发功能可以架本地服务器, 把之前买的域名绑上之后, 再把一直吃灰的 macbook 翻出来, 这样终于可以在外面访问家里的 Centos 虚拟机了。最近还整了一个树莓派, 用来长时间开机作为 Centos 虚拟机的补充。不过目前遇到一个问题, Centos 上开启了 Cisco AnyConnect VPN 之后就没法从外面 ssh 进来, 目前还没想到什么好的解决方法 O_o 这一年从头到尾就是沿着 “发现问题” - “找别人写好的指南” - “遇到了指南里没有的情况” - “痛苦的自己摸索” - “成功解决/我是废物” 这样一个轨迹, 在解决一个个问题的路上过去了。
