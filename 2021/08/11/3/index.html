拿树莓派做了一个git备份仓库和博客副本

发表于 2021-08-11 分类于 技术 ， Linux ， git

拿树莓派做了一个git备份仓库，在push到github的时候会同步推送到本地的树莓派。顺便用nginx在树莓派上架设了一个博客副本, 在树莓派的仓库设置了一个hook, 只要本地的git仓库收到push过来的文件, 就会在pi的public文件夹下pull。

结果忘记给git账户写入权限，折腾了老半天，得先把pi和git加到同一个用户组去.
